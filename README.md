# Outreachy Round 32 - Microtask 1 (Wikimedia Brasil)

Hi! I am **Deeksha**. This repository contains my submission for the first microtask of the **Lusophone technological wishlist proposals** project.

##  Task Description (T418285)
The goal of this task was to create a JavaScript script that processes a JSON array of Wikimedia articles and displays them in a human-readable format within an HTML page.

### Features Implemented:
* **JSON Parsing:** Iterated through the `data` array using `forEach`.
* **Date Formatting:** Converted ISO dates (e.g., `2021-09-13`) to a friendly format (e.g., `September 13, 2021`) using `toLocaleDateString`.
* **DOM Manipulation:** Dynamically created `<p>` elements to display the results in the `#results` div.

##  How to Run
1. Clone this repository or download the `Intern.html` file.
2. Open `Intern.html` in any modern web browser (Chrome, Firefox, Edge).
3. You will see the list of articles and their creation dates formatted as requested.
